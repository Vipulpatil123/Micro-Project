# Student Enrollment Form using JsonPowerDB

## Title of the Project
Student Enrollment Form using JsonPowerDB

---

## Description
This project is a Student Enrollment Form developed using HTML, JavaScript, and JsonPowerDB (JPDB).  
The application allows users to store, retrieve, and update student records in the database using Roll No as the primary key.

The form follows the rules defined in the micro-project problem statement:
- Only Save, Update, and Reset buttons are used.
- Data validation is performed to avoid empty fields.
- The system automatically switches between Save mode and Update mode based on the primary key.

---

## Benefits of using JsonPowerDB
- High performance and fast data processing
- Schema-free database structure
- REST API based communication
- Lightweight and easy to integrate
- Real-time database operations
- Beginner friendly and easy to learn

## Scope of Functionalities

- Validate input fields (no empty values allowed)
